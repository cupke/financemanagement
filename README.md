# FinTrack
                                                                                                                                       
  > Self-hosted веб-приложение для учёта личных финансов с мультивалютностью по курсам ЦБ РФ, бюджетированием и экспортом данных.      
                                                                                                                                       
  Проект разрабатывается в рамках выпускной квалификационной работы по направлению **09.03.04 «Программная инженерия»** (СКФУ,         
  Ставрополь, 2026).                                              
                                                                                                                                       
  ## Что умеет (план MVP)                                                                                                              
   
  - Учёт счетов, категорий, доходов/расходов и переводов между счетами в разных валютах.                                               
  - Загрузка официальных курсов ЦБ РФ из публичного XML-фида.     
  - Бюджеты по категориям с уведомлением о превышении.                                                                                 
  - Сводные отчёты за период и экспорт в CSV/Excel.               
  - Регистрация и аутентификация (JWT, Argon2id).                                                                                      
  - Развёртывание одной командой `docker compose up -d`.
