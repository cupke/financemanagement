# Changelog

Все значимые изменения в проекте документируются в этом файле.
Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
проект придерживается [семантического версионирования](https://semver.org/lang/ru/).

## [1.0.0] — 2026-06-17

Первый стабильный релиз FinTrack — self-hosted веб-приложения для учёта личных финансов.

### Добавлено
- Учёт счетов, доходов, расходов и переводов между счетами
- Категории доходов/расходов с древовидной иерархией и управлением из интерфейса
- Месячные бюджеты по категориям с прогрессом и статусом
- Повторяющиеся операции с ленивой автогенерацией пропущенных
- Мультивалютность с пересчётом по официальным курсам ЦБ РФ (стратегия cache-aside)
- Кросс-валютные переводы между счетами в разных валютах
- Аналитика и отчёты с графиками, фильтрами по счёту и периоду
- Главный дашборд: общий капитал, расходы за месяц, топ-категории, последние операции, бюджеты
- Экспорт операций в CSV с учётом текущих фильтров
- Аутентификация (JWT), регистрация, подтверждение email, восстановление и смена пароля
- Тёмная тема интерфейса (авто + ручной переключатель)
- Открытый REST API (39 эндпоинтов, OpenAPI 3.0, Swagger UI на `/docs`)
- Развёртывание через Docker / docker-compose

### Безопасность
- Хеширование паролей Argon2id
- Изоляция данных между пользователями (ответ 404 при доступе к чужим ресурсам)
- Удаление аккаунта вместе со всеми данными (152-ФЗ, право на забвение)
